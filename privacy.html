This extension does not collect, transmit, sell, or share any personal or sensitive user data. All processing happens locally in your browser, on your own computer. Files you create are saved only to your device through the browser's standard download feature. We (the developer) have no server and receive none of your data.

1. Who we are

This Privacy Policy describes how the browser extension Web Page & AI Chat Archive ("the Extension", "we", "us") handles information. The Extension is a local archiving tool that saves web pages and AI chat conversations to your own computer.

2. Information the Extension processes

The Extension only processes information when you explicitly activate it (by clicking its toolbar icon, choosing a context-menu item, or pressing Save in its panel) on the current tab. It does not run in the background collecting data. The information processed on your device includes:

  • Web page content — the HTML of the active tab and the contents of its linked CSS files and images, used to build a self-contained snapshot (FULL.html, VIEW.html, MHTML, or Markdown).
  • AI chat content — on supported AI chat sites (ChatGPT, Claude, Gemini, Perplexity, Grok), the text of the visible conversation on the active tab, used to build a Markdown transcript (CHAT_MD.md).
  • Page metadata — the URL and title of the active tab, and your browser's User-Agent string, written into an optional log file (LOG.txt) for your own reference.
  • Your settings and custom parsing rules — your chosen file types, language, and any selector rules you configure, stored locally in your browser.

3. How information is used

The single purpose of the Extension is to let you archive pages and AI chats to your own device. Processed information is used only to:

  • Generate the snapshot / transcript / log files you selected; and
  • Save those files to your computer via the browser's standard download function.

The information is never used for advertising, tracking, profiling, analytics, or any purpose unrelated to producing the files you asked for.

4. Storage, sharing, and transmission

  • No transmission to us or third parties. The Extension does not send your browsing data, page content, chat content, or settings to any server operated by the developer or any third party.
  • Local storage only. Your settings and custom rules are stored using the browser's local storage on your device. They are removed if you remove the Extension (or clear its data).
  • Files stay on your device. All generated files are written only to your local Downloads location.
  • Resource fetching during a save. To produce a self-contained snapshot, the Extension may fetch images and stylesheets referenced by the page you are saving, directly from their original web addresses. These requests go to those resource hosts (the same ones your browser already loaded for that page), not to the developer.
  • Optional "remote rules" feature. If — and only if — you manually enter an https:// URL for a parsing-rules file, the Extension will fetch that JSON file from the address you provided. No page content or personal data is sent with that request. This feature is off unless you explicitly use it.

1. 운영 주체

본 방침은 브라우저 확장 프로그램 웹페이지 + AI 채팅 아카이브("본 확장", "당사")가 정보를 어떻게 처리하는지 설명합니다. 본 확장은 웹페이지와 AI 채팅 대화를 사용자 본인의 컴퓨터에 저장하는 로컬 아카이빙 도구입니다.

2. 본 확장이 처리하는 정보

본 확장은 사용자가 현재 탭에서 직접 실행할 때만(툴바 아이콘 클릭, 컨텍스트 메뉴 선택, 패널에서 저장 실행) 정보를 처리합니다. 백그라운드에서 데이터를 수집하지 않습니다. 기기에서 처리되는 정보는 다음과 같습니다.

  • 웹페이지 콘텐츠 — 활성 탭의 HTML, 그리고 자체 포함 스냅샷(FULL.html, VIEW.html, MHTML, Markdown)을 만들기 위한 연결된 CSS·이미지의 내용.
  • AI 채팅 콘텐츠 — 지원 대상 AI 채팅 사이트(ChatGPT, Claude, Gemini, Perplexity, Grok)에서 활성 탭에 표시된 대화 텍스트. Markdown 대화록(CHAT_MD.md) 생성에 사용됩니다.
  • 페이지 메타데이터 — 활성 탭의 URL과 제목, 브라우저의 User-Agent 문자열. 사용자 참고용 로그 파일(LOG.txt)에 선택적으로 기록됩니다.
  • 사용자 설정 및 사용자 지정 파싱 룰 — 선택한 파일 형식, 언어, 사용자가 구성한 선택자 룰 등. 브라우저 로컬에 저장됩니다.

3. 정보의 이용 목적

본 확장의 유일한 목적은 사용자가 페이지와 AI 채팅을 본인 기기에 저장할 수 있게 하는 것입니다. 처리된 정보는 다음 용도로만 사용됩니다.

  • 사용자가 선택한 스냅샷 / 대화록 / 로그 파일 생성;
  • 해당 파일을 브라우저의 표준 다운로드 기능으로 사용자 컴퓨터에 저장.

해당 정보는 광고, 추적, 프로파일링, 분석 등 사용자가 요청한 파일 생성과 무관한 목적으로는 결코 사용되지 않습니다.

4. 저장·공유·전송

  • 당사 또는 제3자로 전송하지 않음. 본 확장은 사용자의 탐색 데이터, 페이지 콘텐츠, 채팅 콘텐츠, 설정을 개발자나 제3자가 운영하는 어떤 서버로도 전송하지 않습니다.
  • 로컬 저장만 사용. 설정과 사용자 룰은 기기의 브라우저 로컬 storage에 저장되며, 확장을 삭제(또는 데이터 삭제)하면 함께 제거됩니다.
  • 파일은 기기에만 저장. 생성된 모든 파일은 로컬 다운로드 위치에만 기록됩니다.
  • 저장 시 리소스 가져오기. 자체 포함 스냅샷을 만들기 위해, 저장 대상 페이지가 참조하는 이미지·스타일시트를 해당 원본 주소에서 직접 가져올 수 있습니다. 이 요청은 (브라우저가 이미 그 페이지를 위해 로드한) 리소스 호스트로 가며, 개발자에게 가지 않습니다.
  • 선택적 "원격 룰" 기능. 사용자가 https:// 주소를 직접 입력한 경우에 한해, 본 확장은 해당 주소에서 파싱 룰 JSON 파일을 가져옵니다. 이 요청에는 어떤 페이지 콘텐츠나 개인정보도 포함되지 않습니다. 사용자가 명시적으로 사용하지 않는 한 이 기능은 동작하지 않습니다.
